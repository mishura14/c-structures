# c-structures
основные структуры данных языка C и алгоритмы сортировки

#Структуры данных
```
-Vector
-Stack
-Single-Linked List
-Double-Linked List
-Tree
-String
-bitVector
```
#Алгоритмы
```
-Bubble Sort
-Quick Sort
-Merge Sort
-Selection Sort
```
#Project Structures

c-structures
```
|
|--structures #структуры данных
|--algorithms #алгоритмы
|--math #математические функции
|--tests #тесты
```

#Сборка и запуск
```
mkdir build
cd build
cmake ..
cmake --build . #имя компилируемого файла 
```
#компиляция тестов
```
cmake --build . tests_имя компилируемого файла
```
